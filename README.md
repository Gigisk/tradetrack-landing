# TradeTrack Landing Page

Professional trade journaling and copy trading platform for futures traders.

## Features

- **Trade Journaling** - Automatic trade import with detailed P&L tracking
- **Trade Copier** - Copy trades to multiple accounts in real-time
- **Webhook Bot** - Connect TradingView alerts to your broker
- **Real-time Sync** - Auto-sync every 30 seconds

## Supported Platforms

- NinjaTrader
- Tradovate
- TopStep (ProjectX)
- Kinetick (Data Feed)

## Deployment

This site is deployed on GitHub Pages.
